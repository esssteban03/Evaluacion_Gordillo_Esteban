# Proyecto de Landing Page Educativa — Coursera Clone

## Descripción
Esta página es una landing page de una plataforma educativa tipo `Coursera`, creada como proyecto de evaluación. El objetivo es presentar cursos, beneficios y servicios formativos con un diseño limpio y responsive.

## Contenido del proyecto
- `index.html`: estructura de la página con navegación, hero, carrusel, beneficios, opciones, sección de conocenos, CTA y formulario de contacto.
- `css/general.css`: variables de color, reset básico, estilos globales, navbar, botones, footer y media queries generales.
- `css/index.css`: estilos específicos para el hero, carrusel, beneficios, cartelera, conocenos, CTA y formulario.
- `img/`: imágenes del hero, carrusel, tarjetas y sección de conocenos.

## Temática del proyecto
La temática principal es:
- educación en línea,
- cursos y especializaciones,
- certificaciones profesionales,
- estudios universitarios a distancia,
- flexibilidad para aprender desde cualquier lugar.

## Cómo usarlo
1. Abre `index.html` en el navegador.
2. Cambia los textos e imágenes para adaptar la página a tu idea de educación o academia.
3. Si quieres cambiar colores, edita las variables en `css/general.css` dentro de `:root`.
4. Si deseas actualizar el hero con otra imagen, reemplaza `img/hero.jpg` con una imagen relevante.
5. Prueba el diseño en diferentes anchos para verificar responsividad.

## Cómo adaptar el contenido
- `hero`: cambia el título, la descripción y el texto del botón.
- `beneficios`: actualiza los 3 bloques con las ventajas del servicio educativo.
- `cartelera`: modifica los títulos y descripciones de las 4 tarjetas.
- `conocenos`: escribe una breve historia del proyecto o de la academia.
- `contacto`: deja el formulario como punto de contacto o solicitud de información.

## Por qué funciona este proyecto
- usa Bootstrap para la navbar y el carrusel,
- aplica Flexbox en beneficios y en la sección de conocenos,
- usa CSS Grid en la sección de opciones,
- está construido mobile-first con media queries para tablet y desktop,
- emplea variables CSS para manejar la paleta de colores.

## Recomendaciones para entrega
- No modifiques la estructura HTML si no es necesario.
- Cambia solo texto, imágenes y colores para adaptar el proyecto.
- Verifica la vista en móvil, tablet y desktop con las herramientas del navegador.
- Mantén el código limpio y evita agregar librerías extras.
